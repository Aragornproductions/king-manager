# King Manager

A free, animated file manager with 20 themes and a password-protected Locker.

## Build instructions
See the setup guide provided by Claude for full step-by-step instructions
on building this app using GitHub Codespaces (no PC required).

## Features
- 20 full themes (gaming, neon, royal, nature, pastel, and more)
- Animated file browser (grid/list, smooth tap + scroll animations)
- Password/PIN-protected private Locker with optional fingerprint unlock
- Splash screen with the King Manager logo
- Auto-build pipeline (GitHub Actions builds the APK on every push)
